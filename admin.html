⚠ Error de permisos Firebase — Solución rápida

Las reglas de tu base de datos bloquean la lectura del listado de equipos. Para arreglarlo:

1. Ve a console.firebase.google.com → tu proyecto → Realtime Database → pestaña Rules

2. Reemplaza todo el contenido por esto y haz clic en Publicar:

{ "rules": { ".read": true, ".write": true } }

Nota: esto permite acceso público de lectura/escritura. Para mayor seguridad en producción puedes usar reglas más específicas.
